lib_sconf.LimitedSizeDict

lib_sconf.LimitedSizeDict#

class lib_sconf.LimitedSizeDict(*args, **kwargs)[source]#

Modified ordered-dictionary which has its size limited by size_limit.

__init__(*args, **kwargs)[source]#

See dict for the general documentation on dictionaries and collections.OrderedDict for documentation about ordered dictionaries. This custom implementation sets a limit to the size of the dictionary by popping the oldest stored item when the size_limit has been reached.
